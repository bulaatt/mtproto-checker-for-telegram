## Install

Requires Node.js 16.20.2+.

```bash
npm install -g tgproxy
tgproxy
```

Package: https://www.npmjs.com/package/tgproxy

Version: v1.0.0

## Notes

- Refreshes MTProto proxy lists from GitHub sources
- Accepts `tg://proxy` and `https://t.me/proxy` links
- Includes English and Russian interface languages
- Supports macOS, Windows, and Linux through npm
