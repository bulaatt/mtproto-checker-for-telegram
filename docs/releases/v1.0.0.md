# MTProto Checker for Telegram v1.0.0

First public GitHub release of MTProto Checker for Telegram.

## What This Release Is

- Node.js desktop CLI release for `macOS`, `Windows`, and `Linux`
- Interactive terminal workflow for refreshing proxy lists, selecting input files, and checking MTProto proxies
- Bilingual interface with English and Russian catalogs

## Release Assets

- `mtproto-checker-for-telegram-1.0.0-macos.tar.gz`
- `mtproto-checker-for-telegram-1.0.0-windows.zip`
- `mtproto-checker-for-telegram-1.0.0-linux.tar.gz`

These are portable archives for a Node-based desktop CLI. They are not installer packages and still require Node.js on the user machine.

## Requirements

- Node.js 16.20.2+
- Internet access for live checks
- A `.txt` file with MTProto proxy links if you do not want to use the built-in GitHub refresh flow

## Main User Flow

1. Install Node.js `16.20.2+`.
2. Extract the archive for your platform.
3. Run `npm install`.
4. Run `npm start`.
5. Choose a language on first run.
6. Refresh GitHub sources or select a proxy list.
7. Run the check and review `~/.tgproxy/data/runtime/working_proxies.txt` on macOS/Linux or `%AppData%\tgproxy\data\runtime\working_proxies.txt` on Windows.
